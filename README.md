# World Cup Datastack

A data pipeline project for World Cup 2026 data — ingestion, transformation, and analytics.

## Project Structure
- `docker/` — containerization configs
- `landing/` — raw/landing zone data
- `producers/` — data ingestion producers
- `schemas/` — data schemas
- `wc2026_dbt/` — dbt models for transformation
- `.github/workflows/` — CI/CD pipelines
